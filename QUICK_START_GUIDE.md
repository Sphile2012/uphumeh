# ⚡ Quick Start Guide - Get Running in 30 Seconds!

## 🚀 Instant Setup (No Configuration Required)

Your Phume Instagram clone works immediately with zero setup!

### 1. Start the App (10 seconds)
```bash
npm run dev
```
✅ App starts at http://localhost:8080

### 2. Login with ANY Credentials (10 seconds)
- **Email**: `your@email.com` (or any email)
- **Password**: `yourpassword` (or any password)
- **Click**: "Sign In" → Enter details → Login!

### 3. Start Using All Features! (10 seconds)
- ✅ **Upload Photos**: Create → Select image → Share
- ✅ **Create Stories**: Home → Click "+" on your profile → Upload
- ✅ **Edit Profile**: Profile → Edit Profile → Change picture
- ✅ **Like & Comment**: Heart and comment on any post
- ✅ **Follow Users**: Visit profiles → Follow button

## 🎯 Key Features to Try

### 📸 Photo Upload
1. **Go to Create** (+ button in navigation)
2. **Upload Image** (drag & drop or click)
3. **Apply Filters** (7 Instagram-style options)
4. **Add Caption** with hashtags
5. **Share** to your feed

### 📱 Stories
1. **Home Page** → Click "+" on your profile picture
2. **Choose**: Photo, Video, or Text Story
3. **Add Text** overlays and backgrounds
4. **Share** to your story (24-hour expiration)

### ✏️ Profile Editing
1. **Profile Page** → "Edit Profile"
2. **Change Picture** (camera icon)
3. **Update Info** (username, bio, website)
4. **Save Changes** → Instant update

### 💬 Social Features
- **Like Posts**: Click heart (smooth animation)
- **Comment**: Type in comment box
- **Follow**: Visit profiles → Follow button
