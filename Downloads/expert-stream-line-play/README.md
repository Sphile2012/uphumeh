# Netflix Clone

A Netflix-inspired streaming UI built with React, Vite, Tailwind CSS, and Framer Motion.

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Framer Motion
- React Router v6
- TanStack React Query
- Radix UI / shadcn components

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Start the dev server: `npm run dev`

## Build

```
npm run build
```

## Deploy

The project is configured for Netlify. Push to your connected repo or run:

```
npx netlify-cli deploy --prod
```
