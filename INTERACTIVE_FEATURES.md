## 📱 How to Use

### Upload Photos
1. Go to Create page (+ button)
2. Click "Select from computer" or drag & drop
3. Choose image/video (max 10MB)
4. Apply filters and adjust brightness
5. Add caption with hashtags
6. Add location (optional)
7. Click "Share" to post

### Change Profile Picture
1. Go to your profile
2. Click camera icon on profile picture
3. Select new image
4. Picture updates instantly

### Interact with Posts
- **Like**: Click heart or double-tap image
- **Comment**: Type in comment box and press "Post"
- **Share**: Click share button for native sharing
- **Save**: Click bookmark to save post
- **More Options**: Click three dots for additional actions

### Follow Users
1. Visit any profile
2. Click "Follow" button
3. Button changes to "Following" when successful
4. Use three dots menu to unfollow, share, or report
